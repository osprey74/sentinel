## What's New / 新機能・変更点

### English

#### UI Refinements
- **Compact layout** — Widget width reduced from 320px to 220px for a slimmer footprint
- **Clock & Calendar split** — Analog clock and calendar now occupy separate stacked sections (previously side-by-side), with the clock centered in its section
- **PC Metrics vertical list** — Changed from 2×2 grid to a single-column list (CPU / MEM / DISK / NET stacked vertically)
- **Weather date labels** — Each forecast day now shows the date in `MM/DD` format between the precipitation probability and the day-of-week label
- **Header cleanup** — Removed the three decorative dots to the left of the "Sentinel" title
- **Section spacing** — Increased vertical padding between sections for better readability
- **Memo bottom margin** — Aligned the memo section's bottom margin with the horizontal margin (14px)

#### Fixes
- **Clock drift fix** — Analog clock now syncs to the wall-clock second boundary, eliminating the ~1-second lag behind the system clock

### 日本語

#### UI 調整
- **コンパクト化** — ウィジェット幅を 320px から 220px に縮小し、より省スペースに
- **時計・カレンダーの分離** — アナログ時計とカレンダーを横並びから上下の独立セクションに分割（時計はセクション内で中央寄せ）
- **PC Metrics 縦一列化** — 2×2 グリッドから CPU / MEM / DISK / NET の縦一列リストに変更
- **天気予報の日付表示** — 各日の降水確率と曜日ラベルの間に `MM/DD` 形式の日付を追加
- **ヘッダー整理** — 「Sentinel」タイトル左の3つの装飾ドットを削除
- **セクション間の間隔調整** — セクション間の縦方向パディングを広げて視認性を改善
- **メモ下マージン調整** — メモセクションの下マージンを横マージン (14px) と揃えた

#### 修正
- **時計のずれを修正** — アナログ時計を秒境界に同期するよう修正し、システム時計との約1秒の遅延を解消
