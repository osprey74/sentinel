## What's New / 新機能・変更点

### English

#### New Features
- **Memo section** — New MEMO section below SELF-HOSTED with Markdown editor
  - Double-click the memo box to open an editor dialog
  - Full Markdown support (headings, lists, bold/italic, code, links, blockquote, etc.)
  - Content persists in localStorage
  - Box height auto-adjusts to content

---

### 日本語

#### 新機能
- **メモセクション** — SELF-HOSTED の下に Markdown 編集対応のメモセクションを追加
  - メモボックスをダブルクリックで編集ダイアログを表示
  - Markdown 完全対応（見出し、リスト、太字/斜体、コード、リンク、引用等）
  - 内容は localStorage に永続化
  - コンテンツに応じてボックスの高さが自動伸縮
